**Description:**
In this proyect we learn how get more security in our devices such as the disks and use DEBIAN 13. This ensures that if the physical machine is stolen, the data remains a useless wall of encrypted noise without the correct passphrase. You will verify this architecture using low-level system commands to prove the layers are correctly stacked.
Then we learn how to save passwords in a ecrypted database called keepass xc, also keepass, help us to calculate the entropy of each passwords, the entropy means the security of the password, the entropy depends on the special characters that you have in you passwords.
Then we learn and study the protect information, we know about two new definitions, hash and cracking.
Hash is like a digital fingerprint for a file. We use a tool called Hashcat to act like a security auditor. 
Then we use PrtonMail, that is a graphic interface for GnuPG and ProtonMail is more different than Mailvelope.
ProtonMail Helps us to send encrypt messagess for other people.

**Objetives**
Implement a security linux in our devices.
Apply entropy in our password for more security.
Analyze the Avalanche Effect in cryptographic hashes (SHA-256) and perform password audit tests using Hashcat.
Send verify infromation with a sign and using good practices.
